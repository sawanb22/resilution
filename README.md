# BitcoinInfo — Informational Landing (Next.js)

This is a simple informational landing site about Bitcoin built with Next.js, Tailwind CSS, and Framer Motion for subtle scroll animations.

Quick start (PowerShell):

```powershell
cd "c:\Users\bhard\Desktop\next.js"
npm install
npm run dev
```

Notes:
- This is a frontend-only informational site. Not financial advice.
- Install dependencies (Next.js, React, Tailwind, Framer Motion) using the command above.
